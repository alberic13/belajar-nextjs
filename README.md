# LKP Exotic - Beauty Salon Landing Page & LMS Foundation

Project ini dibuat sebagai media pembelajaran untuk mendalami berbagai aspek modern web development menggunakan Next.js, mulai dari arsitektur kode, keamanan aplikasi, hingga implementasi animasi interaktif.

## 🎯 Tujuan Project

Project ini berfokus pada:
1. **Struktur & Arsitektur Next.js**: Mempelajari tata letak file, routing, server/client components, dan integrasi database yang efisien.
2. **Implementasi Keamanan (Security)**: Menerapkan praktik terbaik dalam autentikasi, enkripsi password, perlindungan route, dan pengamanan data.
3. **Animasi & Interaktivitas**: Menghadirkan user experience yang premium dengan transisi halus dan animasi modern menggunakan GSAP, Framer Motion, dan Three.js.

Projek ini berupa **Landing Page untuk LKP (Lembaga Kursus dan Pelatihan) Salon Kecantikan**, yang dirancang dengan skalabilitas tinggi agar nantinya dapat dikembangkan menjadi platform **LMS (Learning Management System)** penuh untuk pelatihan kecantikan online.

---

## 🛠️ Tech Stack & Library

Aplikasi ini dibangun menggunakan teknologi mutakhir:

- **Framework**: [Next.js](https://nextjs.org/) (dengan React 19)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animasi**:
  - [GSAP](https://gsap.com/) & `@gsap/react`
  - [Framer Motion](https://www.framer.com/motion/)
  - [Three.js](https://threejs.org/) (untuk interaksi 3D)
- **Database & ORM**:
  - [Drizzle ORM](https://orm.drizzle.team/)
  - [MySQL2](https://github.com/sidorares/node-mysql2)
- **Keamanan & Validasi**:
  - [BCryptJS](https://github.com/dcodeIO/bcrypt.js) (Hashing password)
  - [Jose](https://github.com/panva/jose) (JWT/Token management)
  - [Zod](https://zod.dev/) (Validasi skema data)
